What is DOM and what is the purpose of DOM?

DOM means Document Object Model. It is an application programming interface for HTML. It render web page to tree structure, where objects represent part of the document.

Purpose of DOM: DOM provide us access to manipulate the HTML page dynamically. It also allow us to add, remove, modify and change styles and classes in a web page. So we can say that, DOM is very powerful and flexible tool to create interactive web pages.

How will you select HTML elements using DOM? Name the DOM methods.

There are many ways in to select elements in a DOM using selectors. They are:

1. getElementById()

2. getElementsByClassName()

3. getElementsByTagName()

4. querySelector()

5. querySelectorAll()

What is event bubble?

Event bubbling is a way of event propagation in the Document Object Model. When we click the specific element, event is triggered on that element. Then the event is bubble up and propagate to its all parents element and again event is trigger on all the parents element up to the root. By using stopPropagation() method we will stop this process.

What is a callback function and why will you use it?

A callback function is a function which is executed after another function has finished execution. And it also passed as an argument to another function. It is also Known as anonymous function.

Reasons to use callback functions: To respond many types of actions we can use a callback function to handle event. Click button and submit form is the example.
